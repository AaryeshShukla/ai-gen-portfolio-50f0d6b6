# Modern Blog Platform

## Overview
This is a modern blog platform homepage designed with a creative grid layout and sophisticated serif typography. The application features a search bar, categories tab menu, responsive posts grid, and a newsletter subscription widget.

## Features
- **Responsive Design**: Fully responsive layout that adapts to different screen sizes.
- **Search Functionality**: Instant filtering of posts based on user input.
- **Categories Menu**: Easy navigation through different categories of posts.
- **Newsletter Subscription**: Simple input for users to subscribe to updates.
- **Featured Articles**: Highlighted posts with read-time and author badges.

## Technologies Used
- HTML
- CSS
- JavaScript
- Google Fonts (Lora)

## Setup Instructions
1. Clone the repository or download the files.
2. Open `index.html` in your web browser to view the application.
3. Ensure you have an internet connection to load Google Fonts.
